## 2. English Description

`codesaur/indoraptor` is a **multi-layered**, **full-featured CMS** PHP web framework built on PSR standards (PSR-3, PSR-7, PSR-15).

The framework operates in two layers - **Web** (public website) and **Dashboard** (admin panel) - and works together with other packages in the codesaur ecosystem.

### Key Features

- ‚úî PSR-7/PSR-15 middleware-based architecture
- ‚úî JWT + Session authentication
- ‚úî RBAC (Role-Based Access Control)
- ‚úî Multi-language support (Localization)
- ‚úî CMS modules: News, Pages, Files, References, Settings
- ‚úî MySQL / PostgreSQL / SQLite support
- ‚úî Twig template engine
- ‚úî OpenAI integration (moedit editor)
- ‚úî Image optimization (GD)
- ‚úî PSR-3 logging system
